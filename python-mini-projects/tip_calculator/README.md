\# Tip Calculator



\## Description

This is a Python program that calculates total bill including tip and splits it among members. 

It also allows different contribution for adults and children.



\## Features

\- Add tip to bill

\- Split among members

\- Adult \& child calculation

\- Custom percentage for child payment



\## How to Run

python tip\_calculator.py
